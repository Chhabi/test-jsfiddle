Adding a Ground Overlay to the Map

This example adds a ground overlay, which displays an image showing the allied occupation sectors in Berlin after the Second World War, with the image stretched across a specified geographic area.

The file berlin-sector.png is based on https://en.wikipedia.org/wiki/Allied-occupied_Germany#/media/File:Occupied_Berlin.svg and is Copyright Wikipedia user Stefan-XP (Stefan@XpofPc.de). Licensed under CC BY-SA 3.0 (http://creativecommons.org/licenses/by-sa/3.0/)
